# 💝 生日大冒险 - 马里奥风格RPG游戏

一个超级马里奥风格的横版平台跳跃游戏，专为你老婆准备的浪漫生日惊喜！

## 🎮 游戏特色

- **经典横版跳跃**：类似红白机超级马里奥兄弟的玩法
- **可爱火柴人角色**：真人照片头像 + 可爱火柴人身体
- **完美移动控制**：
  - 🖥️ 电脑端：键盘方向键/WASD + 空格跳跃
  - 📱 手机端：屏幕虚拟按键控制
- **三大关卡（每关玩法不同）**：
  - 💖 关卡1：寻找钥匙碎片 - 收集3个碎片，回答问题获得"爱之钥匙"
  - 🔐 关卡2：开启封印之门 - 用钥匙打开门，回答问题获得"守护之盾"
  - 🛡️ 关卡3：决战BOSS - 用盾牌抵挡BOSS弹幕攻击
- **丰富元素**：
  - ⭐ 收集星星得分
  - 💖 爱心道具恢复生命
  - 🔑 钥匙碎片（关卡1特有）
  - 🚪 封印之门（关卡2特有）
  - ❓ 问号箱回答问题获得武器
  - 👾 踩扁敌人（敌人速度已调慢）
  - 🔮 BOSS弹幕攻击（关卡3特有）
- **武器系统**：
  - 🔑 爱之钥匙 - 打开封印之门
  - 🛡️ 守护之盾 - 抵挡BOSS攻击
  - 上一关的武器是破解下一关的关键！
- **精美动画**：
  - 火柴人走路动画
  - 武器显示
  - 敌人巡逻
  - 道具浮动效果
  - 云朵背景滚动
  - 弹幕攻击特效
  - 结局烟花特效
- **游戏时长**：约2-3分钟

## 🕹️ 操作说明

### 电脑端
- ⬅️➡️ **方向键** 或 **A/D**：左右移动
- ⬆️ **空格键** 或 **W**：跳跃
- 💡 **技巧**：助跑跳跃可以跳得更远！

### 手机端
- ⬅️➡️ **左右按钮**：控制移动方向
- ⬆️ **跳按钮**：跳跃
- 💡 **技巧**：同时按住移动+跳跃可以跳得更远！

## 🎯 游戏目标与流程

### 第一关：💖 寻找钥匙碎片
1. **收集碎片** 🔑 - 在平台上找到3个金色的钥匙碎片
2. **回答问题** ❓ - 收集齐3个碎片后，触碰问号箱回答爱情问题
3. **获得钥匙** 🔑 - 答对后获得"爱之钥匙"武器
4. **到达终点** 🚩 - 找到终点旗帜进入下一关

### 第二关：🔐 开启封印之门
1. **使用钥匙** 🔑 - 用第一关获得的爱之钥匙打开封印之门
2. **穿过封印** 🚪 - 门打开后才能继续前进
3. **回答问题** ❓ - 触碰门后面的问号箱回答问题
4. **获得盾牌** 🛡️ - 答对后获得"守护之盾"武器
5. **到达终点** 🚩 - 前往BOSS关

### 第三关：🛡️ 决战时光守护者
1. **躲避弹幕** 🔮 - BOSS会发射紫色弹幕攻击你
2. **盾牌防御** 🛡️ - 有盾牌时被弹幕击中不会受伤
3. **跳跃攻击** ⚔️ - 从上方跳到BOSS头上进行攻击
4. **击败BOSS** 🐉 - 攻击10次击败BOSS
5. **获得祝福** 🎉 - 完成游戏查看生日祝福

## 💡 核心机制

- **武器传承系统**：每关获得的武器是破解下一关的关键！
  - 没有钥匙无法打开第二关的门
  - 没有盾牌很难在第三关抵挡弹幕
- **不同闯关模式**：
  - 第一关：收集玩法
  - 第二关：解谜玩法
  - 第三关：战斗玩法

## 💡 游戏技巧

### 基础技巧
- 🏃 **助跑跳跃**：移动中跳跃比原地跳得更远
- 🦘 **二段跳**：踩到敌人后会弹起，可以连续跳跃
- 💚 **保护生命**：看到爱心道具记得收集
- 🎯 **踩头攻击**：从上方落到敌人/BOSS头上才能攻击

### 关卡技巧
- 🔑 **第一关**：优先收集所有钥匙碎片，不要急着往前冲
- 🚪 **第二关**：记得带着钥匙去开门，否则无法通过
- 🛡️ **第三关**：有盾牌可以无视弹幕伤害，大胆攻击BOSS

### 高级技巧
- ❓ **必答问题**：问号箱会给3颗星+1条命+武器，一定要答对
- 👾 **敌人利用**：踩敌人可以得星星，还能当跳板
- ⏱️ **速通挑战**：熟练后可以挑战2分钟内通关
- 🎮 **无伤挑战**：不损失任何生命完成游戏

## 🖼️ 替换主角头像（重要！）

游戏中的主角是**火柴人身体+真人头像**的可爱造型！

### 方法1：使用本地图片（推荐）

1. 准备一张你老婆的照片（最好是正方形或圆形头像）
2. 将照片命名为 `avatar.jpg` 或 `avatar.png`
3. 把照片放在与 `index.html` 同一个文件夹中
4. 打开 `index.html`，找到第118行左右：

```javascript
playerAvatar.src = 'data:image/svg+xml...';  // 默认卡通头像
```

5. 改成：

```javascript
playerAvatar.src = 'avatar.jpg';  // 使用本地图片
```

### 方法2：使用在线图片

1. 上传照片到图床（如 imgur.com、postimg.cc 等）
2. 复制图片直链
3. 修改代码：

```javascript
playerAvatar.src = 'https://你的图片链接.com/image.jpg';
```

### 头像效果

- 头像会自动调整为圆形
- 大小约36x36像素，适合游戏显示
- 左右移动时头像会跟随方向转动
- 配合可爱的火柴人身体，非常萌！

## 🐛 常见问题

**Q: 头像没有显示怎么办？**
A: 确保图片文件名和路径正确，或者使用完整的在线图片链接。默认会显示可爱的卡通头像。

**Q: 找不到钥匙碎片怎么办？**
A: 钥匙碎片是金色的，有特殊的钥匙形状。在平台上方跳跃探索，HUD会显示收集进度（x/3）。

**Q: 问号箱打不开？**
A: 第一关需要先收集齐3个钥匙碎片；第二关需要先获得爱之钥匙。检查HUD显示的武器和碎片数量。

**Q: 封印之门打不开？**
A: 必须先完成第一关获得爱之钥匙。触碰门时会提示是否有钥匙。

**Q: BOSS的弹幕太难躲？**
A: 这就是守护之盾的作用！有盾牌时被弹幕击中不会受伤。确保在第二关获得了盾牌。

**Q: 跳不到高台怎么办？**
A: 记得助跑！移动中跳跃可以跳得更远更高。

**Q: 怎么打败敌人？**
A: 从上方跳到它们头上，就像马里奥踩蘑菇一样。敌人速度已调慢，更容易踩到。

**Q: 手机上操作不灵敏？**
A: 确保触摸屏幕底部的虚拟按键，可以同时按住移动和跳跃。

**Q: 想增加难度怎么办？**
A: 修改游戏参数：提高敌人速度、增加BOSS血量、减少生命值、提高BOSS攻击频率。
